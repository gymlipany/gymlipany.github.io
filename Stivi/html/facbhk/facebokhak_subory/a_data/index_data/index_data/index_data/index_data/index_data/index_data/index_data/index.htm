Táto stránka sa zobrazí, keď služba Google automaticky rozpozná žiadosti prichádzajúce z vašej počítačovej siete, ktoré zrejme porušujú Zmluvné podmienky. Platnosť tohto zablokovania vyprší krátko po zastavení odosielania týchto žiadostí. Dovtedy budete musieť pri používaní našich služieb zadávať znaky z obrázka CAPTCHA.

Táto aktivita mohla byť odoslaná škodlivým softvérom, doplnkom prehliadača alebo skriptom, ktorý odosiela automaticky vygenerované žiadosti.
